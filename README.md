## 📝 Abstract & Contributions
Raw-data streaming in edge-intelligence systems faces a severe computational trilemma involving precision, latency, and energy consumption. This project utilizes **Persistent Sheaf Laplacians** to shift the paradigm from raw data transmission to highly localized, geometry-aware feature extraction on microcontrollers. 

**Key Contributions:**
1. **Algorithmic Compression:** 1D symmetric distance matrix flattening reduces topological graph construction memory by 50%.
2. **Computational Acceleration:** An Early-Terminated Union-Find algorithm halts topological boundary reduction early, enabling execution latencies securely under 8 ms.
3. **Clinical Validation:** Evaluated on the MIT-BIH and PPG-DaLiA datasets, proving that localized sheaf features isolate physiological anomalies from heterophilic motion artifacts.
